About ChromeKiwi

ChromeKiwi is a lightweight desktop companion that wanders around your screen, reacts to clicks, and adds a bit of joy to your workday. Built with modern web technologies, it runs on Mac, Windows, and Linux via a simple Chrome or Edge extension.

Features

Installation Instructions

  1. Download the ZIP file and extract to a folder.
  2. Open your browser's chrome://extensions (or edge://extensions).
  3. Enable Developer mode in the top-right corner.
  4. Click Load unpacked and select the extracted folder.
  5. Kiwi Pet will appear and start roaming your desktop!

Note: Remove other conflicting desktop pet extensions for best experience.
